# 服务器规则

## 一、基本守则

1. **互相尊重** — 禁止辱骂、挑衅、恶意骚扰其他玩家。
2. **禁止破坏** — 未经允许不得破坏、窃取他人建筑与物品。
3. **禁止作弊** — 严禁使用外挂、恶意漏洞。
4. **禁止刷屏** — 请勿频繁发送无意义消息或广告。

## 二、建筑相关

- 公共设施（刷怪场、村民交易所等）请标明归属。
- 如需圈地，请查看[插件说明中的领地插件](#plugins)。

## 三、惩罚机制

| 违规行为 | 首次 | 再次 |
|---------|------|------|
| 言语辱骂 | 警告 | 禁言 1 小时 |
| 破坏建筑 | 回档 + 警告 | 封禁 3 天 |
| 使用外挂 | 永久封禁 | — |

> 封禁申诉请联系管理员 QQ，并提供相关证据。

## 四、注意事项

- 服务器定期备份，但请 **自行备份重要建筑**。
- 如发现 BUG 或漏洞，请第一时间报告管理员。
