## 四、重载事件 (OnReload)

当 TShock 执行 `/reload` 命令时，`OnReload` 事件被触发，执行以下操作：

```csharp
private void OnReload(TShockAPI.Hooks.ReloadEventArgs e)
{
    AntiCheat.LoadConfig();              // 重新加载反作弊配置
    AntiCheat.LoadProjConfig();          // 重新加载弹幕配置
    ItemConfigHandler.LoadItemConfig();  // 重新加载物品配置
    ProjDetection.RefreshRestrictedProjectiles();  // 刷新违禁弹幕列表
    ItemDetection.RefreshRestrictedItems();        // 刷新违禁物品列表
    ItemDetection.StartAutoScan();       // 启动自动扫描
    PromotionManager.LoadConfig();       // 重新加载权限提升配置
    
    TShock.Log.ConsoleInfo("[TSWeb] 反作弊配置已重新加载");
}
```

**重载内容总结**：
- ✅ 反作弊配置（AntiCheat）
- ✅ 弹幕配置（ProjConfig）
- ✅ 物品配置（ItemConfig）
- ✅ 违禁弹幕列表刷新
- ✅ 违禁物品列表刷新
- ✅ 自动扫描重启
- ✅ 权限提升配置

## 五、Dispose 生命周期

当插件被卸载或服务器关闭时，`Dispose` 方法执行清理操作：

### 清理顺序

```csharp
protected override void Dispose(bool Disposing)
{
    if (Disposing)
    {
        // 1. 取消重载事件订阅
        GeneralHooks.ReloadEvent -= OnReload;
        
        // 2. 子模块清理
        PlannedOff.Dispose();
        BugFixes.Dispose(this);
        AutoRegister.Dispose(this);
        ItemRestrict.Dispose();
        OnlineData.Dispose();
        SSELogger.Dispose();
        RuntimeHooks.Dispose();
        BossLimit.Dispose();
        ItemDetection.StopAutoScan();
        BypassHelper.UnregisterPermissionHook();
        
        // 3. 清理聊天命令
        CleanupChatCommands();
        
        // 4. 清理 REST API 路由
        CleanupRestApiRoutes();
    }
    base.Dispose(Disposing);
}
```

### REST API 路由清理（CleanupRestApiRoutes）

清理所有 TSWeb 注册的 REST API 路由，通过反射访问 `Rests.Rest` 类的私有 `commands` 字段，移除匹配的路由：

```csharp
var commandsField = typeof(Rests.Rest).GetField("commands",
    BindingFlags.NonPublic | BindingFlags.Instance);

}
```
