## 导入 Dify 指南

### 方式一：单个文件导入（推荐）

1. 打开 Dify 平台 → **知识库**
2. 点击 **创建知识库** → **导入现有文档**
3. 选择本目录下的 Markdown 文件（可多选）
4. 设置索引方式（推荐设置见下方）
5. 点击 **确认导入**

### 方式二：批量文件夹导入

1. 将整个 `tsweb-docs/` 目录打包为 ZIP
2. 在 Dify 中创建知识库 → **导入** → 选择 ZIP 文件
3. Dify 会自动解压并识别所有 Markdown 文件
4. 设置索引方式后确认导入

### 推荐索引设置

| 设置项 | 推荐值 | 说明 |
|--------|--------|------|
| 索引方式 | **高质量** | 针对技术文档，确保检索精度 |
| 分段方式 | **自定义** | 按 Markdown 标题分段（# ##） |
| 分段标识符 | `#` / `##` / `###` | 保持文档层级结构 |
| 分段最大长度 | 1000-2000 tokens | 避免过长或过短 |
| 分段重叠长度 | 100-200 tokens | 保证上下文连贯 |
| 检索策略 | **混合检索** | 关键字 + 向量检索结合 |
