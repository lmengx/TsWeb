### 文档清单
本知识库包含以下7个文档：

### 导入 dify 步骤

#### 1. 准备工作
- 确保您有 dify 平台的访问权限
- 准备好账号和密码

#### 2. 创建知识库
```bash
1. 登录 dify 平台
2. 进入"知识库"页面
3. 点击"创建知识库"
4. 填写知识库名称：TShock Documentation
5. 填写描述：TShock Terraria 服务器管理工具技术文档
```

#### 3. 上传文档
```bash
1. 在知识库详情页，点击"上传文件"
2. 选择所有 .md 文档文件
3. 等待文件上传完成
4. 点击"开始处理"
```

#### 4. 配置向量化
```bash
分段设置：
- 分段模式：自动分段
- 分段标识符：一级标题（#）
- 最大分段长度：1000 tokens

索引设置：
- 索引模式：高质量模式
- Embedding 模型：选择合适的向量模型
- 启用重排序：是
```

#### 5. 创建应用
```bash
1. 进入"应用"页面
2. 创建新应用
3. 选择"对话型应用"
4. 关联 TShock 知识库
5. 配置提示词
```
