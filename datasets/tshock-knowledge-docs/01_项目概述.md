# TShock 项目概述

## 项目简介

TShock 是一个功能强大的 Terraria 服务器工具箱，为 Terraria 服务器和社区提供全面的管理和增强功能。它是一个开源项目，采用 GPL-3.0 许可证。

### 核心定位
TShock 是 Terraria Server API 的一个插件，通过扩展 Terraria 服务器功能，提供企业级的服务器管理能力。

### 版本信息
- 当前版本代号：Profoundly Collaborative (3.11)
- 开发框架：.NET 9
- 编程语言：C#
- API 版本：2.1

## 核心功能

### 1. 反作弊系统
- Bouncer：反作弊和反利用系统
- 物品禁令系统（Item Bans）
- 投射物禁令系统（Projectile Bans）
- 方块禁令系统（Tile Bans）

### 2. 服务器端角色系统（SSC）
- 服务器端角色存储
- 角色数据持久化
- 角色数据上传和同步

### 3. 权限管理系统
- 基于组的权限管理
- 细粒度权限节点
- 用户账户管理
- 权限继承机制

### 4. 区域保护系统
- 区域定义和管理
- 建筑保护
- 区域权限控制

### 5. REST API 接口
- 服务器远程管理
- RESTful API 接口
- Token 认证机制

### 6. 其他核心功能
- 玩家管理（踢出、封禁）
- 传送点系统（Warps）
- 服务器备份
- GeoIP 地理定位
- 多语言支持（i18n）
- 服务器白名单
- 日志系统

## 技术架构

### 整体架构
```
┌─────────────────────────────────────┐
│         Terraria Server             │
├─────────────────────────────────────┤
│      Terraria Server API            │
├─────────────────────────────────────┤
│          TShock Plugin              │
│  ┌───────────────────────────────┐  │
│  │      TShockAPI Core           │  │
│  │  ┌─────────────────────────┐  │  │
│  │  │  Modules & Handlers     │  │  │
│  │  ├─────────────────────────┤  │  │
│  │  │  Database Layer         │  │  │
│  │  ├─────────────────────────┤  │  │
│  │  │  REST API Layer         │  │  │
│  │  └─────────────────────────┘  │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### 核心组件
1. **TShockAPI** - 核心 API 库
2. **TShockLauncher** - 服务器启动器
3. **TShockPluginManager** - 插件管理器
4. **TShockInstaller** - 安装程序
5. **TerrariaServerAPI** - Terraria 服务器 API

### 数据存储
- 支持多种数据库：SQLite, MySQL, PostgreSQL
- 配置文件：JSON 格式
- 日志系统：文本日志和数据库日志

### 依赖关系
- .NET 9 SDK
- Terraria Server
- OTAPI（Open Terraria API）
- Newtonsoft.Json
- MaxMind GeoIP
- MySQL 数据库连接器

## 开发指南

### 环境要求
- .NET 9 SDK
- Visual Studio 2022 或 VS Code
- Git

### 构建步骤
```bash
# 克隆项目
git clone https://github.com/Pryaxis/TShock.git --recurse-submodules

# 进入项目目录
cd TShock

# 构建项目
dotnet build

# 运行服务器
dotnet run --project TShockLauncher

# 发布版本
cd TShockLauncher
dotnet publish -r win-x64 -f net9.0 -c Release -p:PublishSingleFile=true --self-contained false
```
