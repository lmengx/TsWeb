## 使用指南

### 导入到 dify
1. 登录 dify 平台
2. 创建新的知识库
3. 上传所有 `.md` 文档文件
4. 配置向量化和检索参数
5. 创建应用并关联知识库

### 最佳实践

#### 文档分割策略
- 建议按章节分割文档
- 每个文档独立处理
- 保留文档标题和结构

#### 检索优化
- 使用语义搜索
- 配置合适的 top-k 值
- 启用重排序功能

#### 提示词优化
```
你是一个 TShock 专家助手，基于提供的知识库文档回答用户问题。

请遵循以下原则：
1. 优先使用知识库中的信息
2. 提供具体的代码示例和命令
3. 引用相关的文档章节
4. 如果信息不足，明确告知用户
```
