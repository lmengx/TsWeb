# 玩家体验插件

## 分类概述

玩家体验插件专注于提升 Terraria 服务器玩家的游戏体验和便利性。涵盖自动队伍分配、自动垃圾桶、智能扫地、自动钓鱼、永久 Buff、生命上限扩展、复活机制、鬼魂观战、限时无敌、速度控制、宵禁、时间同步/加速、复活币、进度礼包、在线奖励、广播系统、投票功能等。这些插件适合**所有类型的服务器**，能够显著提升玩家满意度和服务器活跃度。

### AutoTeam — 自动队伍

- **作者**: 十七、肝帝熙恩
- **依赖**: LazyAPI
- **说明**: 自动分配指定权限组的玩家到特定队伍（红队、蓝队、绿队等）。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/autoteam` | `/at` | `autoteam.toggle` | 开关自动队伍功能 |
| 无 | 无 | `noautoteam` | 拥有此权限不会被自动分配队伍 |

#### 配置要点

> 配置文件位置：`tshock/AutoTeam.zh-CN.json`

```json5
{
  "开启插件": true,
  "组对应的队伍": {
    "guest": "pink",
    "default": "蓝队",
    "owner": "红队",
    "admin": "green",
    "vip": "none"
  }
}
```

- 支持中文和英文队伍名（红队/red、蓝队/blue、绿队/green、黄队/yellow、粉队/pink、无队伍/none）
- 未配置的组玩家可自由切换队伍

### AutoAirItem — 自动垃圾桶

- **作者**: 羽学
- **依赖**: LazyAPI
- **说明**: 帮助玩家自动清理背包中的指定物品（垃圾），支持玩家自定义自清理列表。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/air` | `/垃圾` | `AutoAir.use` | 指令菜单 |
| `/air on` | `/垃圾 on` | `AutoAir.use` | 开启或关闭垃圾桶功能 |
| `/air list` | `/垃圾 list` | `AutoAir.use` | 列出自己的垃圾桶物品 |
| `/air clear` | `/垃圾 clear` | `AutoAir.use` | 清空自己的垃圾桶表 |
| `/air yes` | `/垃圾 yes` | `AutoAir.use` | 手持物品加入垃圾桶表 |
| `/air auto` | `/垃圾 auto` | `AutoAir.use` | 放入垃圾桶位格时自动添加 |
| `/air mess` | `/垃圾 mess` | `AutoAir.use` | 开启或关闭清理消息 |
| `/air ck 数量` | `/垃圾 ck 数量` | `AutoAir.use` | 筛选超过此数量的玩家 |
| `/air add 或 del id` | — | `AutoAir.use` | 添加或移除垃圾桶物品 |
| `/airreset` | `/重置垃圾桶` | `AutoAir.admin` | 清空玩家数据表 |
| `/reload` | 无 | `tshock.cfg.reload` | 重载配置文件 |

#### 配置要点

> 配置文件位置：`tshock/自动垃圾桶.json`

- **重启服务器不删数据**: 启用后数据持久化
- **排除垃圾表**: 白名单 ID，不会被当作垃圾处理
- 每个玩家独立配置自己的垃圾桶列表
- 数据存储于 `tshock.sqlite` 数据库

### AutoClear — 智能自动扫地

- **作者**: 大豆子（Mute适配1447），肝帝熙恩更新
- **依赖**: LazyAPI
- **说明**: 当地面物品数量达到临界值时，自动清扫地面掉落物。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 自动触发，无需指令 |

#### 配置要点

> 配置文件位置：`tshock/Autoclear.zh-CN.json`

- **多久检测一次**: 默认 10 秒
- **智能清扫数量临界值**: 默认 100 个物品触发
- **延迟清扫**: 触发后延迟 10 秒执行
- **清扫分类**: 可分别控制挥动武器、投掷武器、普通物品、装备、时装的清扫
- **不清扫的物品ID列表**: 白名单保护
- **存在事件或BOSS时跳过**: 避免干扰战斗

### AutoFish — 自动钓鱼重置版

- **作者**: ksqeib（维护者）、羽学、少司命
- **依赖**: LazyAPI
- **说明**: 自动钓鱼插件，支持自动收杆、多钩、Buff、额外渔获、消耗模式等。
- **使用前提**: **必须开启 SSC**（ServerSideCharacter），否则无法正常代扣鱼饵。

#### 权限模型

| 权限 | 说明 |
| :-: | :-: |
| `autofish.admin` | 管理员全通 |
| `autofish.common` | 通用白名单，可使用全部玩家指令 |
| `autofish.<feature>` | 功能权限（如 `autofish.fish`、`autofish.multihook`） |
| `autofish.no.<feature>` | 负权限，强制禁用某功能 |

#### 玩家指令

| 语法 | 所需权限 | 说明 |
| :-: | :-: | :-: |
| `/af` | `autofish` | 查看菜单/帮助 |
| `/af status` | `autofish` | 查看个人状态 |
| `/af fish` | `autofish.fish` | 开/关自动钓鱼 |
| `/af buff` | `autofish.buff` | 开/关钓鱼 Buff |
| `/af multi` | `autofish.multihook` | 开/关多钩 |
| `/af hook 数字` | `autofish.multihook` | 设置个人钩子上限 |
| `/af monster` | `autofish.filter.monster` | 开/关不钓怪物 |
| `/af anim` | `autofish.skipanimation` | 开/关跳过上鱼动画 |
| `/af list` | `autofish` | 查看消耗模式指定物品 |
| `/af loot` | `autofish` | 查看额外渔获表 |
| `/af bait` | `autofish.bait.protect` | 开/关保护贵重鱼饵 |
| `/af baitlist` | `autofish.bait.protect` | 查看贵重鱼饵列表 |

#### 管理员指令（需 `autofish.admin`）

| 语法 | 说明 |
| :-: | :-: |
| `/afa` | 查看管理员帮助菜单 |
| `/afa buff` | 全局开/关钓鱼 Buff |
| `/afa multi` | 全局开/关多线模式 |
| `/afa duo 数字` | 设置全局多钩上限 |
| `/afa mod` | 全局开/关消耗模式 |
| `/afa set 数量` | 设置消耗物品数量 |
| `/afa time 数字` | 设置奖励时长（分钟） |
| `/afa add 物品名` | 添加指定鱼饵 |
| `/afa del 物品名` | 移除指定鱼饵 |
| `/afa addloot 物品名` | 添加额外渔获 |
| `/afa delloot 物品名` | 移除额外渔获 |
| `/afa monster` | 全局开/关不钓怪物 |
| `/afa anim` | 全局开/关跳过上鱼动画 |

#### 配置要点

- 配置位于 `resource/config/zh-cn.yml`
- **消耗模式**: 玩家需通过消耗物品兑换钓鱼时长
- **贵重鱼饵保护**: 自动将贵重鱼饵与背包尾部鱼饵交换位置避免消耗
- **多钩限制**: 受全局上限和个人设置双重约束
- **额外渔获**: 管理员可自定义额外渔获列表

### OnlineGiftPackage — 在线礼包

- **作者**: 星夜神花、羽学
- **依赖**: 无
- **说明**: 定时为在线玩家发放随机在线奖励。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/在线礼包` | 无 | `OnlineGiftPackage` | 显示礼包内所有物品的概率表 |
| `/reload` | 无 | — | 自动计算总概率 |

#### 配置要点

> 配置文件位置：`tshock/在线礼包.json`

- **发放间隔**: 默认 1800 秒（30分钟）
- **跳过生命上限**: 高血量玩家无法获取礼包
- **总概率**: 自动根据所有物品概率计算
- **触发序列**: 按顺序循环发放不同礼包内容

---

### RandomBroadcast — 随机广播

- **作者**: 羽学
- **依赖**: 无
- **说明**: 定时随机发送一条广播内容，支持执行指令。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 自动触发，无需指令 |

#### 配置要点

> 配置文件位置：`tshock/随机广播.json`

- **同发数量**: 同时发送多条消息
- **默认间隔**: 最小可达 0.1 秒
- **触发概率**: 每条消息独立配置概率
- **总概率**: 自动计算
- 消息内容以 `/` 或 `.` 开头的会被当作指令执行

---

### AutoBroadcast — 自动广播

- **作者**: Scavenger、Zaicon、GK、肝帝熙恩
- **依赖**: LazyAPI
- **说明**: 定时在服务器中广播消息或执行命令，支持触发词语触发。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 自动触发，无需指令 |

#### 配置要点

> 配置文件位置：`tshock/AutoBroadcast.zh-CN.json`

- **广播列表**: 支持多条广播独立配置
  - **时间间隔**: 计时广播间隔，设为 0 则不启用
  - **延迟执行**: 计时广播延迟时间
  - **广播组**: 限定特定权限组可见
  - **触发词语**: 玩家发送指定词语时触发广播
  - **触发整个组**: 触发词语时向整个组发送广播

### VotePlus — 多功能投票

- **作者**: Cai
- **依赖**: 无
- **说明**: 多功能投票插件，支持踢出、封禁、禁言、清除BOSS、修改时间/天气、自由主题等投票。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/vote kick <玩家>` | 无 | `vote.use` | 投票踢出指定玩家 |
| `/vote ban <玩家>` | 无 | `vote.use` | 投票封禁指定玩家 |
| `/vote mute <玩家>` | 无 | `vote.use` | 投票禁言指定玩家 |
| `/vote clearboss <BOSS名称>` | 无 | `vote.use` | 投票清理指定BOSS（不掉落） |
| `/vote clearevent` | 无 | `vote.use` | 投票清理地图中所有事件 |
| `/vote day` | 无 | `vote.use` | 投票将时间改为白天 |
| `/vote night` | 无 | `vote.use` | 投票将时间改为黑夜 |
| `/vote rain` | 无 | `vote.use` | 投票下雨或停止下雨 |
| `/vote topic <主题>` | 无 | `vote.use` | 自由主题投票 |
| `/vote clearkick <玩家>` | 无 | `vote.use` | 将玩家从踢出状态移除 |
| `/vote kicklist` | 无 | `vote.use` | 查看处于踢出状态的玩家 |
| `/agree` | 同意 | `vote.use` | 同意当前投票 |
| `/disagree` | 反对 | `vote.use` | 反对当前投票 |

#### 配置要点

> 配置文件位置：`tshock/VotePlus.json`

- **最小投票人数**: 默认 3 人
- **投票通过率**: 默认 60%
- 各项投票功能可独立启用/禁用
- 踢出持续时间可配置，默认 600 秒
