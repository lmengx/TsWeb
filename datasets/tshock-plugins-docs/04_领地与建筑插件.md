# 领地与建筑插件

## 分类概述

领地和建筑相关插件解决服务器中的空间管理和建筑辅助需求：

- **领地保护**：玩家圈地保护自己的建筑，设置权限管理访客
- **区域管理**：智能区域触发命令，自动执行服务器逻辑
- **边界显示**：可视化查看领地边界，避免误入和纠纷
- **建筑辅助**：快速铺桥、复制建筑、自动生成基础建设，降低建服门槛
- **小游戏**：基于建筑玩法的竞速比赛
- **迷宫生成**：在游戏世界中随机生成迷宫，支持计时和排行榜

## HouseRegion 圈地插件

### 概述

功能完善的圈地保护插件，玩家可以圈定自己的房屋区域并管理权限。

- **作者**：GK Eustia
- **依赖**：LazyAPI
- **配置文件**：`tshock/HouseRegion.json`

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/house set 1` | house.use | 敲击设置左上角 |
| `/house set 2` | house.use | 敲击设置右下角 |
| `/house clear` | house.use | 重置临时敲击点 |
| `/house allow [玩家] [房屋]` | house.use / house.admin | 添加所有者 |
| `/house disallow [玩家] [房屋]` | house.use / house.admin | 移除所有者 |
| `/house adduser [玩家] [房屋]` | house.use / house.admin | 添加使用者 |
| `/house deluser [玩家] [房屋]` | house.use / house.admin | 删除使用者 |
| `/house delete [屋名]` | house.use / house.admin | 删除房屋 |
| `/house list [页码]` | house.use / house.admin | 查看房屋列表 |
| `/house redefine [屋名]` | house.use / house.admin | 重新定义房屋范围 |
| `/house info [屋名]` | house.use / house.admin | 查看房屋信息 |
| `/house lock [屋名]` | house.use / house.admin | 锁定/解锁房屋 |
| `/house show [屋名]` | house.use | 高亮显示房屋区域 |
| `/house showall` | house.use | 显示所有房屋区域 |
| `/house auto` | house.use | 进入区域时自动显示 |
| `/house tp [屋名]` | house.use | 传送到房屋 |

### 配置要点

```json
{
  "进出房屋提示": true,         // 进入/离开时提示
  "房屋嘴大大小": 1000,         // 最大面积
  "房屋最小宽度": 30,
  "房屋最小高度": 30,
  "房屋最大数量": 1,            // 每个玩家最大房屋数
  "禁止锁房屋": false,
  "保护宝石锁": false,
  "始终保护箱子": false,
  "冻结警告破坏者": true,
  "禁止分享所有者": false,       // 禁止所有者分享
  "禁止分享使用者": false,       // 禁止使用者分享
  "禁止所有者修改使用者": true
}
```

### 权限节点

| 权限 | 说明 |
|------|------|
| house.use | 普通玩家圈地权限 |
| house.admin | 管理员圈地权限 |

## SmartRegions 智能区域

### 概述

当玩家进入指定区域时自动执行预设命令，支持占位符 `[PLAYERNAME]`。

- **作者**：GameRoom（肝帝熙恩 汉化修复）
- **出处**：[GitHub - ZakFahey/SmartRegions](https://github.com/ZakFahey/SmartRegions)
- **依赖**：LazyAPI
- **配置文件**：`tshock/SmartRegions/SmartRegions.sqlite`

### 功能特性

- 玩家进入区域时自动执行指定命令
- 支持占位符 `[PLAYERNAME]`，自动替换为玩家名称
- 支持区域 Z 值优先级（仅最高 Z 值区域生效）
- 冷却时间控制最小触发间隔（毫秒）
- 支持引用文本文件中的多条命令

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/smartregion add <区域> <冷却> <命令>` | SmartRegions.manage | 为区域绑定命令 |
| `/smartregion remove <区域>` | SmartRegions.manage | 移除区域的命令绑定 |
| `/smartregion check <区域>` | SmartRegions.manage | 查看区域详细信息 |
| `/smartregion list [页码] [距离]` | SmartRegions.manage | 列出智能区域 |
| `/replace <参数>` | SmartRegions.manage | 替换区域配置 |

### 命令配置说明

```bash
# 单条命令
/smartregion add 新手区 1000 /give [PLAYERNAME] 100

# 引用文件（文件放在 tshock/SmartRegions/ 目录下）
/smartregion add 新手区 1000 /file welcome.txt
```

### 权限节点

| 权限 | 说明 |
|------|------|
| SmartRegions.manage | 管理智能区域 |

### 更新日志

| 版本 | 要点 |
|------|------|
| v1.4.7 | 更新英文翻译 |
| v1.4.3 | 完善卸载函数 |
| v1.4.2 | 修复初始化顺序错误 |

## BuildMaster 建筑大师小游戏

### 概述

基于 MiniGamesAPI 框架开发的速建比赛小游戏，类似《我的世界》中的速建大比拼。玩家在限定时间内根据主题建造，结束后互相评分并排名。

- **作者**：Bean_Paste，羽学
- **出处**：[bbstr.net](https://www.bbstr.net/r/117/)
- **依赖**：MiniGamesAPI
- **配置目录**：`tshock/BuildMaster/`
- **运行要求**：需开启 SSC 并运行在旅行模式服务器中

### 游戏流程

```
1. 玩家进入房间 → 查看主题投票
2. 投票决定建造主题 → 准备就绪
3. 开始建造（限时） → 使用创造模式
4. 建造结束 → 互相评分
5. 评分完毕 → 公布排名
```

| `/bm ready` | bm.user | 准备/取消准备 |
| `/bm vote 主题` | bm.user | 投票选择主题 |

## CreateSpawn 复制建筑

### 概述

将选定区域的建筑保存为文件，可以在任意位置（头顶/出生点）生成该建筑，支持跨服复制。

- **作者**：少司命，羽学，Eustia
- **依赖**：LazyAPI
- **配置文件**：`tshock/CreateSpawn.json`

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/cb set 1` | create.build.copy | 敲击/放置方块设置左上角 |
| `/cb set 2` | create.build.copy | 敲击/放置方块设置右下角 |
| `/cb save [名字]` | create.build.copy | 保存选定区域为建筑 |
| `/cb spawn [名字]` | create.build.copy | 在头顶生成建筑 |
| `/cb back` | create.build.copy | 还原建筑覆盖的图格 |
| `/cb list` | create.build.copy | 列出所有已保存建筑 |
| `/cb zip` | create.build.copy | 清空建筑并备份为 zip |
| `/cb auto add [名字]` | create.build.copy | 添加到新世界自动生成列表 |
| `/cb auto remove [名字]` | create.build.copy | 从自动生成列表移除 |
| `/cb auto list` | create.build.copy | 查看自动生成列表 |
| `/cb auto clear` | create.build.copy | 清空自动生成列表 |

### 配置要点

```json5
{
  "中心X": 0,
  "计数Y": 0,
  "微调X": -21,
  "微调Y": 50,
  "自动生成建筑": ["出生点", "大厅"]  // 新世界自动生成的建筑
}
```

### 功能特性

- **跨服复制**：建筑文件可跨服务器使用
- **箱子物品复制**：完美还原箱子内物品
- **标牌信息复制**：保留标牌文字内容
- **家具物品还原**：武器架、帽架、人体模特等家具物品
- **自动生成**：新世界可自动生成指定建筑列表
- **压缩存储**：使用 GZipStream 压缩，节省空间

### 权限节点

| 权限 | 说明 |
|------|------|
| create.build.copy | 复制和生成建筑 |

## SpawnInfra 生成基础建设

### 概述

开服时自动在出生点附近生成全套基础建设，包括监狱集群、箱子集群、世界平台/轨道、地狱直通车、微光湖直通车、刷怪场等。高度可配置，适用于大地图。

- **作者**：羽学
- **出处**：[TShockWorldModify](https://github.com/hufang360/TShockWorldModify)
- **依赖**：无
- **配置文件**：自动生成（内嵌默认值）

### 指令表

| 语法 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/rm 数量` | `/基建 数量` | room.use | 在当前位置建小房子（玩家用） |
| `/rm reset` | `/基建 重置` | room.admin | 重置配置，使开服自动基建重新生效 |

### 生成功能一览

| 功能模块 | 说明 |
|---------|------|
| **监狱集群** | 出生点右侧脚下的标准监狱 |
| **箱子集群** | 8层×多列的收纳箱子系统 |
| **世界平台** | 出生点上方的全方位平台 |
| **左海平台** | 左侧海洋多层平台 |
| **世界轨道** | 配套的轨道系统 |
| **地狱直通车** | 出生点到地狱的直通通道 |
| **微光湖直通车** | 通往微光湖的直通车 |
| **刷怪场** | 洞穴层的简易刷怪场（含岩浆/尖球/飞镖机关） |
| **自建微光湖** | 可选的独立微光湖 |

### 配置要点

```json5
{
  "开服自动基建": true,
  "监狱集群": [{           // 自定义监狱样式
    "图格ID": 38,
    "墙壁ID": 147,
    "平台样式": 43
  }],
  "箱子集群": [{           // 收纳系统配置
    "箱子数量": 18,
    "箱子层数": 8
  }],
  "世界/左海平台": [{       // 平台轨道配置
    "世界平台高度": -200,
    "左海平台长度": 270
  }],
  "直通车/刷怪场": [{       // 直通车和刷怪场配置
    "刷怪场清理深度": 200,
    "刷怪场清理宽度": 200,
    "设置刷怪率": true     // 肉山后自动提高刷怪率
  }]
}
```

> **注意**：此插件预设参数仅适用于大地图，中地图需自行调试适配，不适用于小地图。

### 权限节点

| 权限 | 说明 |
|------|------|
| room.use | 使用 /rm 建小房子 |
| room.admin | 管理基建配置 |

## MazeGenerator 迷宫生成器

### 概述

在游戏世界任意位置创建指定大小的随机迷宫，支持加入迷宫挑战、自动计时、排行榜。

- **作者**：Eustia
- **依赖**：LazyAPI
- **配置目录**：`tshock/MazeGenerator/`

### 功能特性

- 随机生成指定大小的迷宫（基于单元格）
- 支持一键重置和重建已有迷宫
- 查看迷宫完整可行路径
- 自动计时和排行榜
- 可配置墙壁/背景墙/油漆颜色

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/maze help` | maze.generate | 显示帮助信息 |
| `/maze build <名称> [大小]` | maze.generate | 生成迷宫（大小=边长×单元格） |
| `/maze join <名称>` | maze.generate | 加入迷宫游戏 |
| `/maze leave` | maze.generate | 退出迷宫游戏 |
| `/maze list` | maze.generate | 列出所有已保存迷宫 |
| `/maze rank [页码]` | maze.generate | 查看排行榜 |
| `/maze del <名称>` | maze.admin | 删除迷宫并清除方块 |
| `/maze path <名称>` | maze.admin | 显示/隐藏迷宫路径 |
| `/maze reset <名称>` | maze.admin | 重置迷宫 |
| `/maze pos <名称> <tl\|bl\|tr\|br>` | maze.admin | 设置迷宫位置（四角） |

### 配置要点

```json5
{
  "默认迷宫大小": 30,
```
