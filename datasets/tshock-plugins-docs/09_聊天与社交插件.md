# 聊天与社交插件

## 分类概述

聊天与社交插件旨在增强 Terraria 服务器的聊天系统功能和玩家社交体验。涵盖多彩聊天、脏话过滤、表情命令、AI 聊天、跨服聊天、留言墙、登录告示牌、装备展示等内容。这些插件适合**社区服、趣味服、社交向服务器**等需要丰富聊天功能的服务器场景。

## 插件详解

---

### RainbowChat — 彩虹聊天

- **作者**: Professor X制作，nnt升级/汉化，肝帝熙恩、羽学更新
- **依赖**: 无
- **说明**: 使玩家每次说话的颜色不一样，支持渐变色和随机色两种模式。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/rc` | `/rainbowchat` | `rainbowchat.use` | 查看菜单 |
| `/rc gradient` | `/rainbowchat 渐变` | `rainbowchat.use` | 切换渐变色的开关 |
| `/rc random` | `/rainbowchat 随机` | `rainbowchat.use` | 切换随机色的开关 |
| `/rc gradient start RRR,GGG,BBB` | — | `rainbowchat.use` | 设置渐变色开始值 |
| `/rc gradient stop RRR,GGG,BBB` | — | `rainbowchat.use` | 设置渐变色结束值 |
| `/rc on` 或 `/rc off` | — | `rainbowchat.admin` | 全局开关 |
| `/rc rswitch` | — | `rainbowchat.admin` | 随机色全局开关 |
| `/rc gswitch` | — | `rainbowchat.admin` | 渐变色全局开关 |

#### 配置要点

> 配置文件位置：`tshock/RainbowChat.json`

- **全局随机色/渐变色开关**: 管理员可分别控制
- **进服自动开启渐变色**: 控制新玩家是否自动启用
- **修改渐变开始/结束颜色**: 设置默认的 RGB 颜色值
- 玩家可使用指令自定义自己的颜色，不会影响全局配置

### DonotFuck — 禁止脏话

- **作者**: Cai / 羽学
- **依赖**: LazyAPI
- **说明**: 检测玩家发言中的敏感词，自动用 `*` 号替代，支持敏感词记录和管理。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/df` | 无 | `DonotFuck` | 指令菜单 |
| `/df list` | 无 | `DonotFuck` | 列出敏感词表（玩家可看） |
| `/df log` | 无 | `DonotFuck.admin` | 开启或关闭敏感词记录 |
| `/df add 词语` | 无 | `DonotFuck.admin` | 添加敏感词 |
| `/df del 词语` | 无 | `DonotFuck.admin` | 移除敏感词 |
| `/reload` | 无 | `tshock.cfg.reload` | 重载配置文件 |

#### 配置要点

> 配置文件位置：`tshock/禁止脏话/禁止脏话.json`

- **每页行数**: 控制 `/df list` 的分页显示
- **记录日志**: 敏感词触发记录到日志文件
- **脏话表**: 敏感词列表，支持自定义添加
- 检测到敏感词后仅用 `*` 替代，不包含封禁逻辑
- **免检权限**: `DonotFuck` 权限的玩家不受检测

### CaiCustomEmojiCommand — 自定义表情命令

- **作者**: Cai
- **依赖**: LazyAPI
- **说明**: 将常用命令映射为表情ID，发送表情即可执行对应命令。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 发送表情即触发命令 |

#### 配置要点

> 配置文件位置：`tshock/CaiCustomEmojiCommand.zh-CN.json`

```json5
{
  "命令列表": [
    { "命令": "/who", "表情ID": 0 },
    { "命令": "/home", "表情ID": 2 },
    { "命令": "/back", "表情ID": 1 }
  ]
}
```

- 不支持跳过权限检查，命令需要标识符 `/` 或 `.`
- 表情ID可在 Terraria Wiki 上查询
- 适合**简化玩家操作**，无需记忆复杂指令

### AIChatPlugin — AI聊天插件

- **作者**: JTL
- **依赖**: 无
- **说明**: 允许玩家通过聊天与 AI 对话系统互动，支持上下文记忆。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| `/ab` | 无 | 向 AI 提问 |
| `/bcz` | 无 | 清除自己的上下文记录 |
| `/bbz` | 无 | 显示帮助信息 |
| `/aiclear` | `aiclear` | 清除所有玩家的上下文记录 |
| `/reload` | `tshock.cfg.reload` | 重载配置文件 |

#### 配置要点

> 配置文件位置：`tshock/AIChatPlugin.json`

- **回答字数限制**: 默认 1000 字
- **回答自动换行字数**: 默认 50 字换行
- **上下文条数限制**: 默认 100 条
- **超时时间**: 默认 100 秒
- **AI设定**: 可自定义 AI 人设和角色
  - 默认设定为"猫娘"角色
- *注意: 需要服务器网络连接正常，AI无法回答敏感话题*

### ChattyBridge — 跨服聊天

- **作者**: 少司命
- **依赖**: LazyAPI
- **说明**: 让多个服务器之间可以互相聊天，实现跨服社交。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 自动转发，无需指令 |

#### 配置要点

> 配置文件位置：`tshock/ChattyBridge.json`

- **是否转发指令**: 控制是否跨服转发以 `/` 开头的指令消息
- **Rest地址**: 目标服务器的 REST API 地址列表
- **服务器名称**: 本服在跨服聊天中显示的名称
- **验证令牌**: 跨服通信验证令牌（非 REST 令牌）
- **消息格式**: 可自定义聊天、加入、离开的消息格式（支持 {0}服务器名 {1}玩家名 {2}消息）

### NoteWall — 留言墙

- **作者**: 肝帝熙恩
- **依赖**: LazyAPI + linq2db
- **说明**: 玩家可以留下和查看留言，支持留言分页、随机查看、管理删除。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/addnote <内容>` | `addnote` | `notewall.user.add` | 留下留言 |
| `/vinote <序号/玩家名字>` | `viewnote` | `notewall.user.view` | 查看留言 |
| `/notewall <页码/help>` | `notewall` | `notewall.user.page` | 留言墙分页 |
| `/rdnote` | `randomnote` | `notewall.user.random` | 随机查看留言 |
| `/upnote <序号> <新内容>` | `updatenote` | `notewall.user.update` | 修改自己的留言 |
| `/delnote <序号/玩家名字>` | `deletenote` | `notewall.admin.delete` | 删除留言（管理员权限） |
| `/mynote` | `mynote` | `notewall.user.my` | 查看我的历史留言 |

#### 配置要点

> 配置文件位置：`tshock/NoteWall.zh-CN.json`

- **个人最大留言数量**: 默认 5 条
- **留言字数限制**: 默认 50 字
- **屏蔽词列表**: 过滤留言中的敏感词汇
- 数据存储于 `tshock.sqlite` 数据库中的 NoteWall 表

### SignInSign — 告示牌登录

- **作者**: Soofa、羽学、少司命
- **依赖**: 无
- **说明**: 玩家进服弹窗显示告示牌，通过告示牌注册登录、获取物品、BUFF、传送等功能。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/gs r` | 无 | `signinsign.setup` | 重设告示牌 |
| `/gs s` | 无 | `signinsign.tp` | 设置告示牌传送点并自动写入配置 |
| 无 | 无 | `sign.edit` | 允许破坏告示牌权限 |

#### 配置要点

> 配置文件位置：`tshock/告示牌登录.json`

- **是否开启注册登录功能**: 控制玩家通过告示牌注册登录
- **记录角色密码**: 是否记录玩家的角色密码
- **点击告示牌功能**: 支持执行指令、给予BUFF、给予物品、传送
- **创建告示牌内容**: 可自定义欢迎语和引导文本
- *注意: 安装此插件后，玩家无法破坏服务器内的所有告示牌（除非有 `sign.edit` 权限）*

### ShowArmors — 展示装备栏

- **作者**: Ak，肝帝熙恩
- **依赖**: 无
- **说明**: 发送自己的装备配置到聊天框，仅显示装备栏的物品信息。

#### 指令

| 语法 | 别名 | 权限 | 说明 |
| :-: | :-: | :-: | :-: |
| `/装备 [玩家]` | `/show` 或 `/zb` | `ShowArmors` | 显示装备 |

#### 配置要点

- 无配置文件
- 仅显示装备栏物品，不包括其他栏位
- `<>` 为必填参数，`[]` 为选填参数

### ItemDecoration — 手持物品浮动消息

- **作者**: FrankV22
- **依赖**: LazyAPI
- **说明**: 玩家切换手持物品时，物品名称以浮动消息形式出现在头顶，并在聊天中显示。

#### 指令

| 语法 | 权限 | 说明 |
| :-: | :-: | :-: |
| 无 | 无 | 自动触发，无需指令 |

#### 配置要点

> 配置文件位置：`tshock/ItemDecoration.zh-CN.json`

分为两大部分：

**物品聊天**（聊天框显示）:
- 可自定义物品颜色和伤害颜色的 RGB 值
- 可独立控制是否显示名称和伤害

**物品切换**（头顶浮动消息）:
- 支持根据物品稀有度设置不同颜色（-1 灰色 ~ 11 紫色）
- 可独立控制是否显示名称和伤害
- 伤害文本支持自定义（默认 "Damage"）
