# 经济系统插件

## 分类概述

经济系统插件围绕 Economics.Core 核心库构建，解决服务器经济生态的完整需求链：

- **货币流通**：玩家通过击杀怪物、挖掘图格获取货币，死亡时可能掉落
- **职业成长**：多职业升级树，升级消耗货币并解锁新能力
- **交易市场**：玩家间物品交易，实现货币与物品的流通
- **商店系统**：使用货币购买商品，支持多货币价格和等级限制
- **技能系统**：通过 JS 脚本引擎实现自定义技能，支持多种触发模式
- **任务系统**：NPC 对话、击杀怪物、物品收集等任务类型
- **物品回收**：将多余物品兑换为货币
- **武器强化**：消耗货币强化武器属性
- **自定义弹幕**：替换/增强武器弹幕效果

## Economics.Deal 交易插件

### 概述

玩家间交易插件，允许玩家将手持物品上架交易市场，其他玩家可以购买。

- **作者**：少司命，千亦
- **依赖**：Economics.Core
- **配置文件**：`tshock/Economics/Deal.json`

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/deal push [价格] [货币类型]` | economics.deal.use | 发布手持物品到交易市场 |
| `/deal list [页码]` | economics.deal.use | 查看交易物品列表 |
| `/deal search [物品名称]` | economics.deal.use | 搜索已发布的交易物品 |
| `/deal buy [ID]` | economics.deal.use | 购买指定交易物品 |
| `/deal recall [ID]` | economics.deal.use | 撤回已发布的物品 |
| `/deal reset` | economics.deal.reset | 重置交易数据 |

### 配置要点

```json
{
  "最大显示页": 10,
  "交易列表": []
}
```

### 权限节点

| 权限 | 说明 |
|------|------|
| economics.deal.use | 使用交易功能 |
| economics.deal.reset | 重置交易数据 |

## Economics.NPC 自定义怪物奖励

### 概述

配置 NPC 专属的货币奖励，支持按输出瓜分和进度限制。

- **作者**：少司命，千亦
- **依赖**：Economics.Core
- **配置文件**：`tshock/Economics/NPC.json`

### 功能特性

- **额外奖励列表**：为特定怪物配置专属货币奖励
- **按输出瓜分**：根据玩家对 BOSS 造成的伤害按比例分配奖励
- **转换率更改**：调整默认击杀奖励的加成系数
- **进度限制**：限制玩家需要达到的进度才可获取奖励

### 配置要点

```json5
{
  "开启提示": true,
  "额外奖励列表": [
    {
      "怪物ID": 4,
      "怪物名称": "克苏鲁之眼",
      "奖励货币": [
        { "数量": 250000, "货币类型": "战利品" }
      ],
      "按输出瓜分": true
    }
  ],
  "转换率更改": {
    "4": {
      "转换率": 1.5,      // 默认奖励的 150%
      "进度条件": ["克脑", "世吞"]
    }
  }
}
```

### 权限节点

无（自动生效）

### 更新日志

| 版本 | 要点 |
|------|------|
| v2.1.0.0 | 修复转换率语义为加成系数，修复额外奖励提示不触发 |

- **作者**：少司命
- **依赖**：Economics.Core
- **配置文件**：`tshock/Economics/RPG.json`

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/rank [职业名]` | economics.rpg.rank | 升级/转职 |
| `/resetrank` 或 `/重置等级` | economics.rpg.reset | 重置职业 |
| `/level reset` | economics.rpg.admin | 管理员重置等级 |

### 配置要点

```json5
{
  "RPG信息": {
    "战士": {
      "聊天前缀": "[战士]",
      "聊天颜色": [0, 238, 0],
      "聊天后缀": "",
      "聊天格式": "{0}{1}{2}: {3}",
      "唯一职业": false,
      "手持武器": [],              // 升级必须手持指定武器
      "进度限制": ["史莱姆王"],
      "升级指令": ["/permabuff 114"], // 升级后执行指令
      "附加权限": ["tshock.item.give"], // 升级后获得的权限
      "升级奖励": [{"物品ID": 4956, "数量": 1}],
      "升级消耗": [{"数量": 500, "货币类型": ""}],
      "父等级": "萌新"               // 前置等级，形成职业树
    }
  },
  "默认等级": "萌新",
  "重置职业执行命令": [],
  "重置职业广播": "玩家{0}重新选择了职业",
  "重置后踢出": false

```

### 进度限制

## Economics.Skill 技能插件

### 概述

基于 JS 脚本引擎（Jint）的技能系统，支持多种触发模式和复杂的弹幕效果。

- **作者**：少司命
- **依赖**：Economics.Core + Economics.RPG
- **配置文件**：`tshock/Economics/Skill.json`

### 触发模式

| 模式 | 说明 |
|------|------|
| **CD** | 冷却结束后自动触发 |
| **主动** | 玩家主动释放 |
| **打击** | 攻击怪物时触发 |
| **击杀** | 击杀怪物时触发 |
| **死亡** | 玩家死亡时触发 |
| **血量** | 血量达到条件时触发 |
| **蓝量** | 蓝量达到条件时触发 |
| **冲刺** | 玩家冲刺时触发 |
| **装备** | 装备特定物品时触发 |
| **跳跃** | 玩家跳跃时触发 |
| **BUFF** | 获得特定 BUFF 时触发 |
| **技能** | 其他技能触发时联动 |
| **环境** | 处于特定环境时触发 |

> **注意**：`击杀`、`击打`、`主动` 三个模式互斥，无法组合使用。

### 指令表

| 语法 | 权限 | 说明 |
|------|------|------|
| `/skill buy [技能索引]` | economics.skill.use | 购买技能 |
| `/skill del [技能索引]` | economics.skill.use | 解绑技能 |
| `/skill ms` | economics.skill.use | 查看已绑定技能 |
| `/skill delall` | economics.skill.use | 解绑手持武器的所有技能 |
| `/skill clear` | economics.skill.use | 解绑所有技能 |
| `/skill reset` | economics.skill.admin | 重置技能 |
| `/skill give [玩家] [技能索引]` | economics.skill.admin | 给予玩家技能（不安全） |
| `/skill del [玩家] [技能索引]` | economics.skill.admin | 移除目标玩家技能 |
| `/skill clearh [玩家]` | economics.skill.admin | 移除目标玩家隐藏技能 |

### 配置要点

```json5
{
  "绑定技能最大数量": 6,
  "单武器绑定最大技能数量": 6,
  "禁止拉怪表": [],
  "禁止伤怪表": [],
  "技能列表": [
    {
      "名称": "测试技能",
      "技能唯一": false,
      "全服唯一": false,
      "隐藏": false,
      "技能价格": [{ "数量": 0, "货币类型": "经验" }],
      "限制等级": [],
      "限制进度": [],
      "限制技能": [],
      "事件循环": { /* 弹幕循环、范围循环、玩家循环 */ },
      "触发设置": {
        "触发模式": ["主动", "CD"],
        "冷却": 0,
        "血量": 0,
        "物品条件": [],
        "Buff条件": [],
        "环境条件": [],
        "技能条件": []
      },
      "执行脚本": "环绕锁敌.js"
    }
  ]
}
```

### 脚本系统

基于 Jint（JavaScript 解释器），支持复杂的弹幕逻辑。所有脚本必须以 `main` 函数为入口：

```javascript
function main(skill, ply, pos, vel, identify) {
    // skill - 技能配置对象
    // ply - 玩家对象
    // pos - 技能释放位置
    // vel - 技能释放方向
    // identify - 弹幕释放ID

    // 内置函数
    SpawnProjtile(ply, pos, vel, 684, 100, 10, ply.Index);
    SendProjectilePacket(index);
    SendPacket(66, ply.Index, life);
    Schedule((args) => { }, 60);
    log("Hello World");
}
```

| 权限 | 说明 |
|------|------|
| economics.skill.use | 使用技能 |
| economics.skill.admin | 技能管理 |

### 概述

自定义武器弹幕替换/增强，支持物品触发和弹幕触发两种模式。

- **作者**：少司命
- **依赖**：Economics.Core + Economics.RPG
- **配置**：`tshock/Economics/Projectile.json`

### 功能特性

- **物品触发**：使用指定物品时发射自定义弹幕
- **弹幕触发**：指定弹幕命中时触发额外弹幕效果
- **等级限制**：弹幕效果可配置等级要求
- **伤害/击退动态跟随**：弹幕属性可随武器动态变化

### 配置要点

```json5
{
  "弹幕触发": {
    "274": {
      "使用物品时触发": true,
      "来自于召唤物": false,
      "本身就是召唤物": false,
      "弹幕数据": [{
        "弹幕ID": 132,
        "弹幕伤害": 80.0,
        "弹幕击退": 10.0,
        "弹幕射速": 15.0,
        "限制等级": []
      }]
    }
  },
  "物品触发": {
    "1327": {
      "弹幕数据": [ ... ],
      "物品使用弹药": false
    }
  }
}
```

### 权限节点

无（自动生效）
