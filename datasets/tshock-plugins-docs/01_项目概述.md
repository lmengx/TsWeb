# TShockPlugin 项目概述

## 项目简介

**TShockPlugin** 是一个由 [UnrealMultiple](https://github.com/UnrealMultiple) 维护的 **TShock 中文插件集合仓库**，致力于收集、整合和持续更新优质的 TShock 插件。该项目是中文 Terraria 社区中最全面的插件资源库。

### 仓库定位

- **名称**: TShockPlugin
- **维护者**: UnrealMultiple（社区组织）
- **仓库地址**: https://github.com/UnrealMultiple/TShockPlugin
- **许可证**: GPL-3.0（开源免费）
- **目标**: 为 TShock 服务器提供一站式插件解决方案

### 核心数据

- **插件数量**: 130+ 个
- **开发框架**: .NET 6
- **适配 TShock**: 5.2+
- **持续更新**: 活跃维护，定期发布新版本
- **多语言支持**: 简体中文、English、Español 等

## 核心特点

### 1. 全面的插件生态
覆盖服务器管理的各个方面，从基础管理到特殊玩法，从经济系统到小游戏，满足各类服务器的需求。

### 2. 统一的依赖体系
- **LazyAPI**: 核心基础库，提供指令注册、配置管理、数据库操作、多语言等基础设施
- **linq2db**: ORM 框架，用于数据库操作
- **Economics.Core**: 经济系统前置库
- **MiniGamesAPI**: 小游戏开发 API

### 3. 便捷的安装方式
推荐使用 **AutoPluginManager (APM)** 一键安装，自动补齐依赖：
```
/apm l              # 列出所有可用插件
/apm i <插件名>     # 一键安装指定插件
/apm u [插件名]     # 检查并更新插件
```

### 4. 活跃的社区支持
- GitHub Issues 反馈 BUG 和建议
- QQ 群实时交流（816771079）
- Crowdin 多语言翻译平台
- 持续的 PR 贡献和代码审查

## 插件依赖体系

### 核心依赖关系

```
TShock 5.2+
  ├── .NET 6
  ├── linq2db (ORM 数据库框架)
  └── LazyAPI (基础库)
        ├── 自动指令注册
        ├── 配置管理 (JSON)
        ├── 数据库工具 (linq2db 封装)
        ├── 多语言支持 (i18n)
        └── REST API 支持
```

### 多级依赖关系

| 依赖层级 | 插件 | 用途 |
|:-------:|------|------|
| **L0** | `linq2db` | 底层 ORM 框架 |
| **L1** | `LazyAPI` | 基础库，大部分插件依赖 |
| **L2** | `Economics.Core` | 经济系统前置 |
| **L2** | `MiniGamesAPI` | 小游戏 API |
| **L2** | `DeltaForce.Protocol` | 三角洲通信协议 |
| **L3** | `Economics.RPG` | 依赖 Economics.Core |
| **L3** | `Economics.Shop` | 依赖 Economics.Core + RPG |
| **L3** | `DataSync` | 进度同步，被 ProgressRestrict 依赖 |

### 重要提示

> **许多插件依赖于 LazyAPI，而 LazyAPI 依赖于 linq2db。**
> 使用本仓库插件前，推荐先安装 `LazyAPI.dll` 和 `linq2db.dll`。
> **强烈推荐使用 AutoPluginManager 安装插件，会自动补齐相应的依赖。**

## 使用注意事项

### 1. 推荐使用 APM 安装
AutoPluginManager 可以自动检测和安装依赖，推荐使用以下命令：
```
/apm i <插件名>
/apm u        # 检查并更新所有插件
```

### 2. 核心依赖必须优先安装
使用本仓库前，请确保已安装以下依赖：
```
/apm i LazyAPI
```
LazyAPI 会自动处理 linq2db 依赖。

### 3. 配置文件路径
每个插件的配置文件位于 `tshock/` 目录下以插件名命名的子目录中：
```
tshock/LazyAPI/config.json
tshock/Economics/config.json
tshock/AutoFish/config.json
```

### 4. 多语言支持
大部分插件支持多语言配置。可以在启动时指定语言：
```
-culture zh     # 中文
-culture en     # 英文
```

### 5. 反馈 BUG
反馈 BUG 时务必按照 GitHub Issue 模板填写，提供详细信息，必要时安装 `pdb` 符号文件以便调试。
