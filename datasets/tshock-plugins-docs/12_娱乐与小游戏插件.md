# 娱乐与小游戏插件

## 分类概述

娱乐与小游戏插件为 TShock 服务器提供各类休闲娱乐功能和迷你游戏体验。这些插件包括类 Among Us 社交推理游戏、玩家决斗系统、增强游戏难度的挑战模式、BadApple 视频播放器、玩家位置随机交换、世界反转、沙尘暴控制、高尔夫奖励、音乐播放器、二维码生成器、合成表查询、奖励箱以及多功能投票系统。这些插件显著增强了服务器的趣味性和玩家互动性。

## 插件详解

### SurvivalCrisis — 类 Among Us 小游戏

**功能说明**

SurvivalCrisis 是一款在 Terraria 中实现类似 Among Us 玩法的社交推理小游戏。玩家被随机分配为"生存者"和"背板者"两种身份，生存者需要收集资源、完成集体任务并在决战中击败 Boss，背板者则需伪装身份、阻挠生存者。游戏需要至少 4 名玩家，持续约一昼夜的游戏时间。

**游戏身份**

- **生存者**：收集资源、提交任务物品、在决战前做好准备击败 Boss
- **背板者**：伪装成生存者、使用 PvP 手段消灭落单生存者、阻挠 Boss 击败

**游戏流程**

1. 玩家进入准备区域，30 秒倒计时后开始
2. 游戏进入夜晚，强制开启 PvP
3. 发生三次"特殊事件"
4. 决战前有投票环节可淘汰玩家
5. 第二天强制传送回出生点，召唤弱化机械骷髅王
6. 生存者击败 Boss 获胜，否则背板者获胜

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/hotreload` | sc.hotreload | 热重载插件 |
| `/scd` | sc.debug | 调试命令 |
| `/sc` | sc.player | 玩家命令 |
| `/sc pch <聊天内容>` | sc.player | 背板者私聊 |
| `/sc task` | sc.player | 查看生存者任务 |
| `/sc st <数字>` | sc.player | 背板者商店 |
| `/sc rank` | sc.player | 查看分数排名 |
| `/sc score` | sc.player | 查看战绩 |
| `/sc nt` | sc.player | 更换称号 |
| `/sc np` | sc.player | 更换前缀 |

**配置要点**

- 配置文件位于 `SurvivalCrisis/CrisisConfig.json`
- 需要配置 7 层地图区域的边界坐标（天空层、地表层、洞穴层、空洞层、迷宫层、迷幻洞穴、地狱层）
- 预设建筑位于 `Islands/` 和 `Spheres/` 文件夹
- 游戏至少需要 4 名玩家才能开始

### PvPer — 决斗系统

**功能说明**

PvPer 实现了玩家间的 PvP 决斗功能，使用 SQLite 数据库存储决斗数据。玩家可以互相邀请进行一对一决斗，系统会记录胜场、连胜等数据。注意：玩家离开竞技场范围会被直接杀死判输。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/pvp add <玩家名>` | pvper.use | 邀请玩家参加决斗 |
| `/pvp yes` | pvper.use | 同意决斗 |
| `/pvp no` | pvper.use | 拒绝决斗 |
| `/pvp list` | pvper.use | 查看排行榜 |
| `/pvp data` | pvper.use | 战绩查询 |
| `/pvp WL` | pvper.use | 查封禁武器表 |
| `/pvp BL` | pvper.use | 查封禁 BUFF 表 |
| `/pvp set 1 2 3 4` | pvper.use | 设置竞技场坐标 |
| `/pvp reset` | pvper.admin | 重置玩家数据库表 |
| `/pvp.BW add <武器名>` | pvper.admin | 封禁指定武器 |
| `/pvp.BB add <增益ID>` | pvper.admin | 封禁指定增益 |

**配置要点**

- 配置文件位于 `tshock/决斗系统.json`
- 需要设置邀请者和受邀者的传送坐标
- 需要设置竞技场的左上角和右下角边界
- 可配置离场扣血、拉回竞技场、拉取范围等参数
- 支持禁用武器表和禁用 BUFF 表

### Challenger — 挑战者模式

**功能说明**

Challenger 是一个增强游戏难度的挑战模式插件。它修改了 BOSS AI、为装备套装和饰品添加被动技能效果，并使玩家受伤时会被抽出血包（血包会给附近的敌怪回血，玩家可拦截血包给自己回血）。支持通过配置文件控制所有怪物的血量倍数。

**套装增强效果（部分）**

| 套装 | 效果 |
|------|------|
| 箭袋饰品 | 永久 BUFF、无限弹药 |
| 挖矿套装 | 永久挖矿 BUFF、连锁挖矿 |
| 暗影套 | 暴击时生成吞噬怪飞弹 |
| 猩红套 | 暴击时从敌怪处吸血 |
| 神圣套 | 击中时召唤光与暗剑气 |
| 甲虫套 | 增加血量上限、伤害治疗队友 |

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/cenable` | 无 | challenger.enable | 启用/解除挑战模式 |
| `/tips` | 无 | challenger.tips | 启用/取消内容提示 |
| `/cf` | 无 | 无 | 切换或触发装备被动技能 |

**配置要点**

- 配置文件位于 `tshock/ChallengerConfig.json`
- 可独立开关挑战模式和 BOSS 魔改
- 可配置怪物吸血比率、吸血上限
- 每个套装的弹幕 ID、伤害、间隔均可独立配置
- 支持配置永久 BUFF 列表和连锁挖矿的图格 ID 表

### BadApplePlayer — BadApple 播放器

**功能说明**

在 Terraria 中使用方块播放 BadApple 视频。插件实现了增量渲染优化和帧数据压缩，可在游戏内用方块阵列播放视频内容。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/badapple help` | badapple.use | 显示帮助信息 |
| `/badapple play <名称> [loop]` | badapple.use | 播放视频（可选循环模式） |
| `/badapple pause <名称>` | badapple.use | 暂停播放 |
| `/badapple resume <名称>` | badapple.use | 继续播放 |
| `/badapple stop <名称>` | badapple.use | 停止播放 |
| `/badapple list` | badapple.use | 列出所有保存的位置 |
| `/badapple playing` | badapple.use | 查看当前播放中的会话 |
| `/badapple pos <名称> <对齐方式>` | badapple.admin | 设置播放位置 |
| `/badapple del <名称>` | badapple.admin | 删除播放位置 |

**配置要点**

- 需要先使用 `/badapple pos` 设置播放位置
- 对齐方式：tl（左上角）、bl（左下角）、tr（右上角）、br（右下角）
- 多人环境下的性能和流畅度可能存在问题

### PlayerRandomSwapper — 玩家位置随机交换

**功能说明**

随机交换玩家的位置。支持多人模式（所有玩家随机交换）和双人模式（随机选取两名玩家交换）。拥有 `playerswap` 权限且不在死亡状态的玩家才会被列入传送列表。

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/swaptoggle as` | allowself | swapplugin.toggle | 切换双人模式自交换 |
| `/swaptoggle en` | enable | swapplugin.toggle | 切换随机位置互换 |
| `/swaptoggle i <秒>` | interval | swapplugin.toggle | 设置传送间隔 |
| `/swaptoggle maxi <秒>` | maxinterval | swapplugin.toggle | 设置最大间隔 |
| `/swaptoggle mini <秒>` | mininterval | swapplugin.toggle | 设置最小间隔 |
| `/swaptoggle ri` | randominterval | swapplugin.toggle | 切换随机间隔 |
| `/swaptoggle swap` | swap | swapplugin.toggle | 切换广播交换信息 |
| `/swaptoggle timer [阈值]` | timer | swapplugin.toggle | 切换广播倒计时 |

**配置要点**

- 配置文件位于 `tshock/PlayerRandomSwapper.zh-CN.json`
- 支持配置总开关、传送间隔、随机间隔范围
- 可切换多人打乱模式/双人模式
- 玩家少于 2 人时不会传送

### Sandstorm — 切换沙尘暴

**功能说明**

使用指令切换沙尘暴的开启和停止，可在配置文件中定义风速值。

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/sd` | /沙尘暴 | sandstorm.admin | 切换沙尘暴开启与停止 |

**配置要点**

- 配置文件位于 `tshock/Sandstorm.json`
- 可配置开启沙尘暴的风速目标值（默认 35）和关闭目标值（默认 20）
- 可开关广播通知

### GolfRewards — 高尔夫奖励

**功能说明**

玩家在高尔夫球洞中击球，根据击球杆数获得相应奖励。支持自定义奖励表，包括物品奖励和执行的命令。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/blockpos` 或 `/物块坐标` | 物块坐标 | 辅助指令，确定方块坐标 |

**配置要点**

- 配置文件位于 `tshock/高尔夫奖励.json`
- 每个球洞可配置坐标、最少/最多杆数
- 支持物品奖励（物品 ID、数量、前缀）和指令执行
- 可设置是否仅限领取一次奖励

### MusicPlayer — 简易音乐播放器

**功能说明**

通过读取音高文件，使用竖琴（harp）、铃铛（bell）或吉他（theaxe）模拟播放音乐。支持简易 MIDI 转换。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/song <歌曲名> [harp/bell/theaxe]` | song | 为自己播放/停止歌曲 |
| `/songall <歌曲名> [harp/bell/theaxe]` | songall | 为所有人播放/停止歌曲 |
| `/songlist` | songlist | 查看歌曲文件列表 |

**配置要点**

- 音乐文件放在 `tshock/Song/` 文件夹
- 音高范围为 C4-C6
- 提供了简易 MIDI 转换脚本和工具

### QRCoder — 二维码生成器

**功能说明**

通过指令在地图上生成由方块构成的二维码图案墙。二维码由晶莹钻石墙、暗影漆和夜明漆构成，边缘有一圈回声墙。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/qr <内容> [尺寸]` | qr.add | 生成二维码 |
| `/qrpos <tl/tr/bl/br>` | qr.add | 设置二维码位置对齐方式 |
| `/qrconf <键> <值>` | qr.add | 设置二维码配置 |

**配置要点**

- 配置文件位于 `tshock/QRCoder.json`
- 可自定义底墙、底颜色、码墙、码颜色
- 支持夜明漆开关和纠错等级设置
- 推荐尺寸为 5

### CaiRewardChest — 奖励箱

**功能说明**

将自然生成的箱子变为每个玩家只能领取一次的奖励箱。玩家点击即可领取，并获得对应的箱子掉落物。支持自定义奖励箱。

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/rcinit` 或 `/初始化奖励箱` | CaiRewardChest.Init | 将全图箱子设为奖励箱 |
| `/rcadd` 或 `/添加奖励箱` | CaiRewardChest.Add | 添加一个奖励箱 |
| `/rcedit` 或 `/编辑奖励箱` | CaiRewardChest.Edit | 编辑一个奖励箱 |
| `/rcdelete` 或 `/删除奖励箱` | CaiRewardChest.Delete | 删除一个奖励箱 |
| `/rcclear` 或 `/清空奖励箱` | CaiRewardChest.Clear | 移除所有奖励箱 |

**配置要点**

- 依赖：linq2db + LazyAPI
- 建议在生成地图后立即使用 `/初始化奖励箱`
- 数据库记录每个玩家对每个箱子的领取状态

### VotePlus — 多功能投票

**功能说明**

提供多种类型的投票功能，包括投票踢出、封禁、禁言、清理 Boss、修改时间、修改天气以及自由主题投票。

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/vote kick <玩家>` | 无 | vote.use | 投票踢出指定玩家 |
| `/vote ban <玩家>` | 无 | vote.use | 投票封禁指定玩家 |
| `/vote mute <玩家>` | 无 | vote.use | 投票禁言指定玩家 |
| `/vote clearboss <BOSS名称>` | 无 | vote.use | 投票清理指定BOSS |
| `/vote clearevent` | 无 | vote.use | 投票清理地图事件 |
| `/vote day` | 无 | vote.use | 投票改为白天 |
| `/vote night` | 无 | vote.use | 投票改为黑夜 |
| `/vote rain` | 无 | vote.use | 投票下雨/停雨 |
| `/vote topic <主题>` | 无 | vote.use | 自由主题投票 |
| `/vote clearkick <玩家>` | 无 | vote.use | 移除踢出状态 |
| `/vote kicklist` | 无 | vote.use | 查看踢出状态玩家 |
| `/agree` | 同意 | vote.use | 同意当前投票 |
| `/disagree` | 反对 | vote.use | 反对当前投票 |

**配置要点**

- 配置文件位于 `tshock/VotePlus.json`
- 可独立启用/禁用每种投票类型
- 可配置最小投票人数和投票通过率
- 踢出持续时间可配置
