# 第三方集成插件

## 分类概述

第三方集成插件用于将 TShock 服务器与外部平台和工具进行连接。包括 QQ 机器人适配插件（CaiBot 和 Lagrange.XocMat）、AI 聊天接口集成、以及数据包调试工具。这些插件扩展了服务器的对外交互能力，实现了游戏内与外部平台的通信和联动。

---

## 插件清单

| 名称 | 功能简述 | 依赖 | 作者 |
|------|---------|------|------|
| CaiBotLite | CaiBot 官方机器人适配插件 | linq2db + GenerateMap | Cai, 羽学, 西江 |
| Lagrange.XocMat.Adapter | Lagrange.XocMat 适配插件 | SixLabors.ImageSharp | 少司命 |
| AIChatPlugin | AI 聊天（接入 AI API） | 无 | JTL |
| CaiPacketDebug | Cai 数据包调试工具 | 无 | Cai |

## 插件详解

### CaiBotLite — CaiBot 官方机器人适配插件

**功能说明**

CaiBotLite 是 CaiBot 官方机器人的服务器适配插件。通过 WebSocket 与 CaiBot 通信，实现 QQ 群与游戏服务器的深度联动。支持白名单管理、进度查询、在线列表、地图查看、远程指令执行等功能。

**功能特性**

- **白名单系统**：QQ 群白名单管理，自动拦截未注册玩家
- **进度同步**：向 QQ 群同步服务器游戏进度（Boss 击杀状态）
- **在线排行**：查询服务器在线时长排行
- **地图查看**：生成并上传地图缩略图
- **远程指令**：通过 QQ 群发送指令到服务器
- **聊天同步**：游戏内聊天与 QQ 群聊天互通
- **Boss 统计**：Boss 击杀次数统计和排行

**指令表**

| 指令 | 别名 | 权限 | 说明 |
|------|------|------|------|
| `/caibotlite debug` | `/cbl debug` | caibot.admin | 调试模式开关 |
| `/caibotlite code` | `/cbl code` | caibot.admin | 生成验证码 |
| `/caibotlite info` | `/cbl info` | caibot.admin | 显示 CaiBot 状态信息 |
| `/caibotlite unbind` | `/cbl unbind` | caibot.admin | 主动解除群绑定 |
| `/caibotlite whitelist` | `/cbl whitelist` | caibot.admin | 开关白名单 |
| `/caibotlite group <群号>` | `/cbl group <群号>` | caibot.admin | 设置白名单提示群号 |

**配置要点**

- 配置文件位于 `tshock/CaiBotLite.json`
- 密钥由系统自动配置
- 可设置白名单开关和白名单拦截提示的群号
- 依赖：linq2db、GenerateMap

### Lagrange.XocMat.Adapter — Lagrange.XocMat 适配插件

**功能说明**

Lagrange.XocMat 适配插件，将 TShock 服务器连接到 Lagrange.XocMat 机器人框架。支持 WebSocket 通信、阻止未注册玩家进入、以及自动重置设置。

**配置要点**

- 配置文件位于 `tshock/Lagrange.XocMat.Adapter.json`
- 依赖：SixLabors.ImageSharp

**配置示例**

```json5
{
  "阻止未注册进入": true,
  "阻止语句": ["未注册禁止进入服务器！"],
  "Socket": {
    "套字节地址": "127.0.0.1",
    "服务器名称": "玄荒",
    "端口": 6000,
    "心跳包间隔": 60000,
    "重连间隔": 5000,
    "空指令注册": ["购买", "抽"],
    "验证令牌": ""
  },
  "重置设置": {
    "删除地图": true,
    "删除日志": true,
    "执行命令": ["/skill reset", "/deal reset"],
    "删除表": ["economics", "regions", "tscharacter", "warps"]
  }
}
```

**配置项说明**

- `阻止未注册进入`：开启后未在机器人注册的玩家无法进入服务器
- `阻止语句`：拒绝玩家进入时显示的提示
- `Socket.验证令牌`：与 Lagrange.XocMat 配置中的令牌保持一致
- `重置设置`：自动重置时执行的命令和删除的数据库表

### AIChatPlugin — AI 聊天插件

**功能说明**

允许玩家通过游戏内聊天与 AI 对话系统进行互动。插件提供简单的命令接口，玩家可通过特定命令或聊天触发词向 AI 提问并接收回答。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/ab` | 无 | 向 AI 提问 |
| `/bcz` | 无 | 清除自己的上下文记录 |
| `/bbz` | 无 | 显示帮助信息 |
| `/aiclear` | aiclear | 清除所有玩家的上下文记录 |
| `/reload` | tshock.cfg.reload | 重载配置文件 |

**配置要点**

- 配置文件位于 `tshock/AIChatPlugin.json`

**配置示例**

```json5
{
  "回答字数限制": 1000,
  "回答自动换行字数": 50,
  "上下文条数限制": 100,
  "超时时间（秒）": 100,
  "对话时显示的AI名字": "猫娘",
  "AI设定": "你是一只可爱的猫娘..."
}
```

**注意事项**

- 确保服务器网络连接正常，以便 AI 能访问必要的 API
- AI 无法回答敏感话题，否则会报错

### CaiPacketDebug — Cai 数据包调试工具

**功能说明**

用于查看客户端和服务器之间的数据包内容。支持分别调试服务端→客户端（S→C）和客户端→服务端（C→S）的数据包。

**指令表**

| 指令 | 权限 | 说明 |
|------|------|------|
| `/cpd` | CaiPacketDebug.Use | 查看调试启用状态 |
| `/cpd stc` | CaiPacketDebug.Use | S→C 数据包调试 |
| `/cpd cts` | CaiPacketDebug.Use | C→S 数据包调试 |

**配置要点**

- 配置文件位于 `tshock/CaiPacketDebug.zh-CN.json`

**配置示例**

```json5
{
  "C->S": {
    "自启动": false,
    "白名单模式": false,
    "白名单模式数据包": [1, 2, 3],
    "排除数据包": [114, 514]
  },
  "S->C": {
    "自启动": false,
    "白名单模式": false,
    "白名单模式数据包": [1, 2, 3],
    "排除数据包": [23]
  }
}
```

**配置项说明**

- `自启动`：服务器启动时自动启用调试
- `白名单模式`：仅显示白名单列表中的数据包
- `排除数据包`：过滤掉指定 ID 的数据包
